# Visual Bible

## Overall look
Live-action-style photoreal/cinematic generation (Higgsfield seedance/kling-class
models), desaturated cool-toned color grade, high contrast, practical-light sources
(streetlamps, CRT glow, flashlights) over any flat/even lighting. 16:9 for full
episodes, 9:16 crops for Shorts.

## Recurring visual motifs
- Analog clocks and watches stopped at 3:17 — always in-frame at least once per
  episode.
- Reflective surfaces (mirrors, dark windows, still water) treated as a threat object,
  never neutral set dressing.
- Radio static / signal-scan overlays as a recurring transition device between scenes.
- A single second of complete stillness (no ambient motion, no sound) whenever the
  3:17 event triggers on-screen.

## Character visual anchors for prompts (keep verbatim across all generations)
- Ethan: "male, late 20s, tired eyes, a pale scar along the left jaw, wearing an
  analog wristwatch frozen at 3:17"
- Maya: "female, late 20s, sharp features, always wearing a silver scarf around her
  neck"
- Lena: "female, late 20s-early 30s, guarded expression, a visible burn scar"
- Noah: "young, quiet demeanor, headphones or a handheld radio scanner nearly always
  present"

## Consistency rule for image/video prompts
Every character-generation prompt in every episode file must restate the anchor
description above verbatim before any scene-specific action — this is what kept
Episodes 1-2 consistent in the original session and must not be dropped.
