# Locations Bible

- **Ethan's apartment** — cramped, radio equipment everywhere, every clock in the unit
  physically stopped/taped over except one he checks obsessively.
- **The radio station (night shift)** — Ethan's workplace, mostly empty overnight,
  long fluorescent-lit corridors, a natural home for "something in the reflection"
  beats.
- **The city at 3:17** — recurring wide establishing shots: streets, all clocks visible
  in frame frozen at the same time.
- **Maya's archive / municipal records office** — where Project Echo's name first
  surfaces (Act 2).
- **Noah's radio den** — cluttered with scanner equipment, introduced Act 2.
- **The Facility** — introduced Act 3, sterile/industrial, mid-century civil-defense
  aesthetic mixed with modern surveillance tech. Culminates in the exit-door finale
  location where Dr. Vail appears.
